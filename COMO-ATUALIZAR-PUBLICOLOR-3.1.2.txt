PUBLICOLOR PCP 3.1.2 — ATUALIZAÇÃO

1. Faça backup do repositório atual.
2. Preserve somente as pastas/arquivos locais:
   - .git
   - .env.local
3. Substitua os demais arquivos pelo conteúdo deste ZIP, diretamente na raiz.
4. Execute:

   corepack enable
   pnpm install --frozen-lockfile
   pnpm validate

5. Envie para o GitHub:

   git add -A
   git commit -m "Publicolor 3.1.2: miniaturas progressivas e indicadores operacionais"
   git push

6. Aguarde o deployment da Vercel.
7. No navegador, use Ctrl + Shift + R.

BANCO DE DADOS
- Não executar SQL.
- Esta versão não altera tabelas, colunas, funções, políticas ou migrations.
