PUBLICOLOR PCP 3.4.3 — COMO ATUALIZAR

Esta versão altera somente o código do sistema. Não execute novo SQL.

1. Preserve na raiz atual:
   - .git
   - .env.local

2. Substitua os demais arquivos pelo conteúdo deste ZIP.

3. Use Node.js 24.x e execute:

corepack enable
pnpm install --frozen-lockfile
pnpm validate

4. Publique:

git add -A
git commit -m "Publicolor 3.4.3: pilhas de subpedidos no Kanban"
git push

5. Após o deploy:
   - computador: Ctrl + Shift + R;
   - PWA/celular: feche e abra o aplicativo novamente.

TESTE PRINCIPAL

- Coloque pelo menos dois subpedidos da mesma OP no mesmo setor e status.
- Confirme que aparecem em uma pilha recolhida.
- Abra a pilha e use as ações individuais.
- Atualize a página e confirme que a pilha voltou recolhida.
- Mova um item para outro setor ou status e confirme que ele saiu da pilha original.
