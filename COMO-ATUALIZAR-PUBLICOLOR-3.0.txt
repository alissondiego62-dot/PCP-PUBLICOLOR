PCP PUBLICOLOR 3.0 — ATUALIZAÇÃO COMPLETA

Este ZIP contém o projeto integral. O conteúdo começa diretamente em app/, components/, lib/ e supabase/.
Não crie uma segunda pasta PCP-PUBLICOLOR dentro do repositório.

ORDEM RECOMENDADA

1. Faça backup do repositório e do Supabase.
2. No SQL Editor do Supabase, execute:
   SQL-ATUALIZACAO-PUBLICOLOR-3.0.sql
3. Na raiz atual do GitHub PCP-PUBLICOLOR, substitua o conteúdo pelos arquivos deste ZIP.
4. Preserve apenas o seu .env.local local. Nunca envie chaves secretas ao GitHub.
5. Execute localmente:
   pnpm install --frozen-lockfile
   pnpm validate
6. Faça commit e push para a branch de produção.
7. Aguarde o deploy da Vercel.
8. Atualize o navegador com Ctrl + F5.
