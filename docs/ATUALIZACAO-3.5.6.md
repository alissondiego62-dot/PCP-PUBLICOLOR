# Publicolor PCP 3.5.6

## PDF e miniaturas

- Todas as páginas convertidas do PDF são enquadradas em uma tela PNG com proporção A4 paisagem.
- O conteúdo é centralizado e redimensionado com `contain`, sem recortar nenhuma parte da página original.
- O usuário pode marcar uma página complementar para ser a miniatura principal do pedido relacionado.
- Quando uma página complementar é escolhida, a página principal original continua disponível na galeria e na impressão.

## Arquivo original

- O PDF selecionado no importador é anexado automaticamente ao Google Drive de cada pedido criado.
- O cadastro manual de pedido possui um campo para selecionar o PDF original da OS.
- O cadastro com PDF exige que a integração do Google Drive esteja conectada.

## Banco de dados

Nenhuma alteração SQL é necessária para esta versão.
