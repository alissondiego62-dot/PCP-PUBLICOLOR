# Publicolor PCP 3.5.1

## Calendário da Agenda

- Datas com pedidos agendados e já posicionados no setor Instalação continuam usando o marcador amarelo.
- Datas com pedidos agendados que ainda estejam fora do setor Instalação recebem marcador vermelho e contador próprio.
- Um mesmo dia pode exibir simultaneamente os dois marcadores.
- O calendário inclui legenda para as duas situações.
- Ao selecionar uma data com inconsistência, o painel lateral exibe um alerta vermelho com a quantidade encontrada.

## Banco de dados

Esta versão não altera tabelas, políticas ou funções. O SQL da versão 3.5.0 Revisado continua válido.
