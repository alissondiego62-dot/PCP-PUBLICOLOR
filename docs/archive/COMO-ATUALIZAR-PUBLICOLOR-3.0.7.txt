PUBLICOLOR 3.0.7 — ATUALIZAÇÃO COMPLETA

1. Faça backup do repositório atual.
2. Preserve a pasta .git e o arquivo .env.local.
3. Apague os demais arquivos do projeto antigo.
4. Extraia todo o conteúdo deste ZIP diretamente na raiz do PCP-PUBLICOLOR.
5. Confirme que app/, components/, public/, supabase/ e package.json estão na raiz.
6. Execute:

   pnpm install
   pnpm validate

7. Envie ao GitHub:

   git add -A
   git commit -m "Publicolor 3.0.7: Kanban móvel e instalação PWA"
   git push

8. Aguarde o deployment da Vercel.
9. No celular, abra https://pcp-publicolor.vercel.app/ e atualize a página.
10. Use o aviso "Publicolor no celular" para instalar. No iPhone/iPad, use Safari > Compartilhar > Adicionar à Tela de Início.

BANCO DE DADOS

Esta atualização não altera o Supabase. Não execute SQL.

VALIDAÇÃO DO KANBAN

- Abra Produção · Kanban em um celular.
- Deslize horizontalmente para avançar setor por setor.
- Teste as setas anterior/próximo.
- Toque nos indicadores para ir diretamente a um setor.
- Confirme que a página continua rolando verticalmente.
- Teste busca e filtros; o indicador de setor deve se adaptar aos setores visíveis.
