PUBLICOLOR PCP 3.2.0 — COMO ATUALIZAR

1. Faça backup do banco e do repositório.
2. Preserve as pastas/arquivos locais:
   - .git
   - .env.local
3. Execute em HOMOLOGAÇÃO:
   SQL-ATUALIZACAO-PUBLICOLOR-3.2.0.sql
4. Execute em HOMOLOGAÇÃO:
   SQL-VALIDAR-PUBLICOLOR-3.2.0.sql
5. Extraia o ZIP diretamente na raiz do projeto.
6. Execute:
   corepack enable
   pnpm install --frozen-lockfile
   pnpm validate
7. Teste:
   - atualização de cada rota sem voltar ao Dashboard;
   - indicadores do Dashboard;
   - edição de nome de material pela atividade;
   - abertura rápida da OS;
   - abas Atividades, Compras e Finalizadas;
   - Google Drive, miniaturas e PWA.
8. Após aprovar a homologação, aplique o SQL no Supabase de produção.
9. Publique:
   git add -A
   git commit -m "Publicolor 3.2.0: dashboard, rotas e otimizacoes"
   git push
10. Após o deploy, use Ctrl + Shift + R. No celular, feche e abra novamente o PWA.
