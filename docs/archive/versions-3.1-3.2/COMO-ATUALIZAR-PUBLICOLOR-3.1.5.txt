PUBLICOLOR PCP 3.1.5 — ATIVIDADES COMPACTAS E SALVAMENTO AUTOMÁTICO

1. Esta atualização não exige novo SQL.
2. Confirme que o banco já possui a estrutura da versão 3.1.4.
3. Preserve na raiz do projeto:
   - .git
   - .env.local
4. Substitua os demais arquivos pelo conteúdo deste ZIP.
5. Execute:

corepack enable
pnpm install --frozen-lockfile
pnpm validate

6. Teste na aba Atividades:
   - recolher e expandir subatividades;
   - copiar a lista principal;
   - copiar uma subatividade;
   - alterar preço e aguardar o indicador de salvo;
   - pressionar Enter no preço para avançar;
   - abrir Nova atividade, digitar um título e pressionar Enter várias vezes;
   - alterar o status da principal e confirmar a aplicação nas subatividades.

7. Publique:

git add -A
git commit -m "Publicolor 3.1.5: atividades compactas e autosave de precos"
git push

8. Após o deploy, faça Ctrl + Shift + R. No celular, feche e abra novamente o PWA para atualizar o cache.
