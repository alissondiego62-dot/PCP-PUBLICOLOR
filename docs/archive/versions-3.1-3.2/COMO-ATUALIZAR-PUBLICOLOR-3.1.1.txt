PUBLICOLOR PCP 3.1.1 — MINIATURAS MAIS RÁPIDAS

1. Preserve somente as pastas/arquivos locais .git e .env.local.
2. Extraia o conteúdo do ZIP diretamente na raiz do repositório PCP-PUBLICOLOR.
3. Execute:

   corepack enable
   pnpm install --frozen-lockfile
   pnpm validate

4. Envie ao GitHub:

   git add -A
   git commit -m "Publicolor 3.1.1: otimizar carregamento das miniaturas"
   git push

5. Aguarde o deploy da Vercel.
6. Entre no sistema como administrador.
7. Abra Configurações > Otimizar carregamento das miniaturas.
8. Clique em Otimizar todas e aguarde a conclusão.

NÃO É NECESSÁRIO EXECUTAR SQL NESTA VERSÃO.
