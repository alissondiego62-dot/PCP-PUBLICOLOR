PUBLICOLOR PCP 3.5.6 - ATUALIZAÇÃO

1. Substitua os arquivos do projeto pela versão 3.5.6.
2. Não é necessário executar novo SQL.
3. Confirme que as variáveis do Google Drive continuam configuradas na Vercel.
4. Execute:

   corepack enable
   pnpm install --frozen-lockfile
   pnpm validate

5. Publique novamente na Vercel.
6. No navegador, pressione Ctrl + F5 para atualizar o Service Worker.

FUNCIONALIDADES DESTA VERSÃO
- PNG de cada página ajustado em A4 paisagem, sem recorte.

- Campo de PDF original no cadastro manual de pedidos.
